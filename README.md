# 2DGE-Protein-Spot-Detection
This repository contains the scripts and pipelines used for NextTopModel's YOLO 2DGE spot object detection model.

## Contents
The repository contains three main contents: and R project directory, a Jupyter notebook containing the synthetic data generation script, and a CellProfiler pipeline that was used as a benchmark for our model. To ensure reproducibility, the R script used to generate the plots used for this project utilizes *renv*. This allows the versions of the packages used in the script to be obtained from the *lockfile* using the *load* function.
